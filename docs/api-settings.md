# Pengaturan API Khusus

Halaman ini digunakan untuk menyimpan konfigurasi koneksi API eksternal dari panel admin.

Field yang tersedia:

- `API Base URL` — endpoint dasar untuk permintaan API.
- `API Key` — token atau kunci autentikasi untuk akses API.
- `Timeout (detik)` — batas waktu maksimum permintaan API.

Kunci database yang digunakan:

- `api_settings.base_url`
- `api_settings.api_key`
- `api_settings.timeout`

Jika API Key sudah tersimpan, admin dapat membiarkan field kosong untuk mempertahankan nilai lama.
