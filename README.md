# Website Absensi Klien — v.1.0.0

Sistem Absensi Digital berbasis web yang dirancang untuk mengelola kehadiran dengan efisien menggunakan teknologi QR Code.

## Fitur Utama
- **Dashboard Analitik**: Pantau kehadiran secara real-time.
- **Manajemen User**: Kelola data siswa, guru, dan staff.
- **Scan QR Code**: Sistem pemindaian cepat untuk absensi.
- **Manajemen Lisensi**: Sistem distribusi lisensi untuk klien SaaS.

## Spesifikasi
- **Framework**: Laravel 11
- **UI Template**: Vuexy Admin Template
- **Database**: MariaDB

## Instalasi
1. Clone repository.
2. Konfigurasi `.env` (Gunakan kredensial dari `docs/bahan.txt`).
3. Jalankan `php artisan migrate`.
4. Jalankan `npm install && npm run build`.
